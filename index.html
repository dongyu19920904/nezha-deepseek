提示词模板宝库

风格模仿写作

模仿特定风格创作宣传文案，融入指定元素和特点

帮我用[周杰伦]的风格写一段[奶茶店]的宣传语，要求：
1. 模仿[周杰伦]的语言特点：
   - 要有他标志性的[特点1]
   - 要用他常用的[特点2]
   - 要有他歌里常见的[特点3]
2. 内容必须包含：
   - [奶茶店]的[特色1]
   - [奶茶店]的[特色2]
   - 一个吸引人来的理由
3. 字数控制在[XX]字以内
4. 最后加一句朗朗上口的口号
适用场景：营销文案、创意写作 创意写作

短视频脚本创作

创作结构完整、修辞丰富的短视频脚本

帮我写一个[短视频]脚本，主题是[主题]，要求：
1. 用上这三种修辞手法：
   - 比喻（至少3个新鲜的比喻）
   - 拟人（让[物品]有人的特点）
   - 排比（用3个相似结构的句子）
2. 视频结构：
   - 开头要在3秒内抓住人眼球
   - 中间要有一个小反转
   - 结尾要有记忆点
3. 适合在[平台]发布
4. 总时长控制在[XX]秒
5. 配上3个表情符号增加趣味性
适用场景：社交媒体、内容营销 视频创作

自问自答学习笔记

创建生动有趣的学习笔记，通过对话形式解释复杂概念

请帮我制作一份[学科]的学习笔记，主题是"[具体知识点]"，要求：
1. 先用夹心饼干写法写一段知识点介绍：
   - 正经介绍知识点
   - 括号里加上通俗解释或记忆窍门
2. 再用精分对话术设计一段问答：
   - 角色A："困惑学生"（提出3个常见疑问）
   - 角色B："耐心老师"（给出清晰解答）
3. 最后用表格总结3个关键知识点和应用场景
4. 整体风格要生动有趣，适合初学者理解
适用场景：学习笔记、知识整理 教育学习

电商产品描述

创作吸引人的电商产品描述，融合多种写作技巧

请帮我为[产品名称]写一份电商产品描述，要求综合使用三种技巧：
1. 夹心饼干写法：
   - 正经描述产品特点
   - 括号里加上买家可能的真实想法
2. 精分对话术：
   - 角色A："犹豫的顾客"（提出3个常见顾虑）
   - 角色B："经验丰富的用户"（分享使用体验）
3. 游戏选择法：
   - 设计2个使用场景的选择
   - 每个选择展示不同的产品优势
4. 总字数控制在[XX]字以内
5. 语言要生动有趣，像朋友推荐一样
适用场景：电商、产品营销 电商营销

活动执行计划书

制定详细的活动执行计划，包含时间表、人员分工和应急预案

帮我为[活动名称]制定一个详细的执行计划，要求：
                                1. 活动基本信息：
                                   - 活动目的：[目的]
                                   - 活动时间：[时间]
                                   - 活动地点：[地点]
                                   - 预计参与人数：[人数]
                                2. 分阶段执行计划（用表格呈现）：
                                   - 前期准备阶段（活动前2周）
                                   - 活动执行阶段（活动当天）
                                   - 后期总结阶段（活动后1周）
                                3. 人员分工（至少包含4个角色）
                                4. 预算规划（分类列出各项支出）
                                5. 应急预案（至少考虑3种突发情况）
                                6. 成功标准（如何评估活动效果）
适用场景：活动策划、项目管理 活动策划

知识觉醒三连招

创新学习方法，将新知识与已有经验联系起来

【知识觉醒三连招】请帮我用创新方式学习[主题]：
                                
                                1. 回忆高光时刻（找到相似经验）：
                                   - 请问我之前是否有过类似的学习经验：[描述你可能有的相关经验]
                                   - 在那次经验中，我成功掌握了什么技能
                                   - 当时是用了什么方法学会的
                                   - 那次经验中最关键的突破点是什么
                                
                                2. 连接新招式（建立知识联系）：
                                   - 请帮我找出[旧知识点]和[新知识点]之间的3个相似之处
                                   - 解释如何把旧知识的学习方法应用到新知识上
                                   - 设计一个练习，把两种知识结合起来
                                   - 找出可能阻碍知识连接的"思维盲点"
                                
                                3. 创造必杀技（知识创新应用）：
                                   - 基于前两步，创造一个独特的学习方法或记忆技巧
                                   - 设计一个只有10分钟的"闪电学习"计划
                                   - 提出一个创新的应用场景
                                   - 给这个"必杀技"起个容易记住的名字
                                
                                请用生动有趣的语言回答，就像在教一个充满好奇心的孩子。
适用场景：学习新技能、知识整合 学习方法

需求表达烧烤架

任务类型 正确打开方式 作死示范
算数题 "证明烤串优惠券哪种划算" "先画坐标系再假设变量..."
编故事 "写哪吒大战WiFi怪的段子，要带四川话" "自由发挥"（得到八股文）
写程序 "自动计算奶茶店排队时间的代码" "先定义变量再写循环..."（程序自爆）

哪吒揪着头发："学个锤子提示语！直接问不就行了？"
太乙的猪猪存钱罐"哐当"吐出硬币："就像烧烤——"

啥是提示语？

"就像你喊'老板来20串微辣鸡翅'！"

  • 要啥：20串鸡翅（指令）
  • 背景：老板知道你是常客（上下文）
  • 别搞错：微辣不是变态辣（期望）

哪吒挣扎："那直接说'给点吃的'不行吗？！"
太乙甩拂尘："就像你要'冰阔落'——不说清楚，老板可能给你倒热开水！"

太乙真人の点AI秘籍

1 说人话

"要AI写检讨？就说：'编个逃课理由，要感动陈塘关百姓的！'
别整'请生成300字情感充沛的叙述文本'这种官腔！"

2 会变通

"就像买奶茶：
'四季春+波霸+少冰+三分糖'（AI秒懂）
'随便来杯好喝的'（等着喝刷锅水吧）"

3 懂规矩

"让AI算零花钱要说：
'这个月还剩200，皮肤打七折时买，烧烤选周二半价！'
别只说'帮我算算钱'！"

三太子の独门心法

1 夹心饼干写法

正经说明（加上通俗解释）
例如："帮我写个请假条（就是那种看了让班主任心软的）"

2 精分对话术

让AI扮演多个角色对话
例如："设计一段对话：哪吒（想逃课）VS太乙真人（严厉教育）"

3 游戏选择法

设计多个选项让AI分析
例如："分析三种逃课理由哪个最可信：肚子疼/看病/家里有事"
